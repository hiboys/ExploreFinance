# Word 文档（doc）品类操作指引

本目录提供 Word 文档（doc）品类的专业操作能力，包括公文、合同、通知、协议书等专业规范化文件的格式套用与美化。

## 功能

- **格式套用**: 将纯文本排版美化并导出为在线文档（Word格式）

## 使用场景

- 创建正式文档（通知、报告、公文、合同等）
- 将纯文本转换为格式与排版美化后的 Word 文档

## 可用模块

### 格式套用模块 (`doc_format`)

将纯文本转换为排版美化后的文档。

## 工作流程

**执行前必须:**

1. **阅读相关文档(`doc/doc_format/README.md`)**
2. **理解工作流程**
3. **执行各步骤**

使用 `tencent-docs` MCP Server 中的 `doc.*` 系列工具执行读写、美化等操作。

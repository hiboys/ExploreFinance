# 【申万宏观汇率研报深度分析报告】

**标题**：地缘冲突中，人民币何以"新高"？  
**作者**：赵伟（首席经济学家）、陈达飞（宏观首席）等  
**机构**：申万宏源证券  
**发布日期**：2026-04-15  
**报告类型**：宏观/汇率专题

---

## 核心结论

**作者观点**：中东地缘冲突背景下，人民币逆势升值创近3年新高，这一"韧性"源于中国能源结构优势+结汇潮延续+美元反弹温和；中长期看，地缘冲突将通过强化新能源出口优势、推动结算多元化和全球资金再配置三条路径，继续支撑人民币升值逻辑。

**我的判断**：⚠️ **部分认同，但存在过度乐观倾向** —— 短期逻辑（能源优势、结汇潮）有数据支撑，但中长期三条升值路径的论证存在较强的假设依赖性，且对风险因素的讨论流于表面。

## 二、看逻辑（推理链条）

### 2.1 作者完整逻辑链

```
【现象】人民币在地缘冲突中逆势升值、创3年新高
    ↓
【短期解释】三大支撑因素
    ├── ① 基本面：中国经济周期企稳 + 能源结构优势凸显
    │       └── 中国油气依赖度27.5% vs 全球55.5%
    │       └── 霍尔木兹依赖度24.7%（低于其他亚洲经济体）
    ├── ② 资金面：结汇潮延续 + 外资未大幅外流
    │       └── 2月结售汇顺差552亿美元（历史次高）
    │       └── 主动型外资撤出美欧但流出中国相对平缓
    └── ③ 外部环境：美元反弹温和
            └── 加息预期不强 + 便利性溢价回落
            └── 美元最高仅反弹2.1%（vs 2022年更大幅度）
    ↓
【中长期判断】地缘冲突将强化升值逻辑
    ├── ① 能源安全溢价 → 新能源出口优势 → 贸易顺差 → 汇率支撑
    ├── ② 结算格局重塑 → 石油美元松动 → 人民币能源贸易结算占比提升
    └── ③ 资金再配置 → 降低美欧集中配置 → 提升中国配置比例
    ↓
【结论】人民币中长期升值逻辑得到强化
```

### 2.2 逻辑漏洞与未论证点

| 环节 | 问题 | 严重程度 |
|------|------|----------|
| **"经济周期底部已出现"** | 断言缺乏数据支撑，与房地产持续下行、消费复苏乏力的现实存在矛盾 | 🔴 高 |
| **能源优势→汇率传导** | 逻辑链条过长，中间环节（能源成本优势→出口竞争力→贸易顺差→汇率）每一步都存在不确定性 | 🟡 中 |
| **新能源出口→汇率支撑** | 假设新能源出口持续增长，但未考虑贸易保护主义、产能过剩、技术替代等风险 | 🟡 中 |
| **石油美元松动** | 过度推断，单一地缘事件难以撼动美元在能源贸易中的主导地位 | 🔴 高 |
| **主权基金增配中国** | 缺乏实证数据，"调查显示"来源不明 | 🟡 中 |

### 2.3 遗漏的重要变量

1. **美联储政策转向风险**：虽然报告认为"加息预期不强"，但若通胀反弹或经济超预期，美联储仍可能转鹰
2. **中美利差**：未讨论中美利差对汇率的影响，这是汇率分析的核心变量之一
3. **资本管制**：未提及中国资本账户开放程度对汇率的影响
4. **房地产风险**：未讨论房地产持续下行对汇率的潜在压力
5. **地缘政治的反向作用**：地缘冲突也可能导致避险情绪升温，反而支撑美元

### 2.4 逻辑陷阱识别

| 陷阱类型 | 具体表现 |
|----------|----------|
| **相关性≠因果性** | 人民币与美元"双强"格局历史上仅出现一次（2021.7-2022.2），样本量过小，难以建立稳定规律 |
| **历史预测未来偏差** | 2021年的"双强"背景（疫情扰动下中国制造优势）与当前（地缘冲突）完全不同，类比不当 |
| **选择性呈现** | 强调能源优势，但未提中国煤炭依赖度高的风险；强调外资未大幅外流，但未提被动型外资流出情况 |
| **过度推断** | 从单一地缘事件推断"石油美元体系边际松动"，缺乏历史依据 |

## 四、综合判断

### 4.1 可采纳部分

| 内容 | 可信度 | 说明 |
|------|--------|------|
| **短期人民币升值的事实** | ✅ 高 | 数据可验证，现象明确 |
| **中国能源结构相对优势** | ✅ 中高 | 油气依赖度数据基本准确，逻辑合理 |
| **结汇潮的支撑作用** | ✅ 高 | 2月结售汇顺差数据真实，历史对比清晰 |
| **美元反弹温和的判断** | ✅ 中高 | 与2022年对比有说服力 |

### 4.2 需验证部分

| 内容 | 验证方法 | 数据来源 |
|------|----------|----------|
| "经济周期底部已出现" | 跟踪PMI、工业增加值、社零等数据 | 国家统计局 |
| 外资流出情况 | 对比主动型与被动型外资流向 | EPFR、Wind |
| 新能源出口份额数据 | 核实统计口径和来源 | 海关总署、行业协会 |
| 主权基金投资意向调查 | 查找原始调查来源 | 相关机构报告 |

### 4.3 存疑部分

| 内容 | 质疑理由 |
|------|----------|
| **石油美元松动** | 过度推断，缺乏实证支撑，历史惯性极强 |
| **人民币能源贸易结算占比提升** | 未提供当前占比数据和提升路径，停留在概念层面 |
| **中长期升值逻辑强化** | 三个支撑因素都是长期趋势，但汇率短期波动更多受利差、资本流动等金融因素影响，报告对这些核心变量讨论不足 |

### 4.4 我的独立判断

**短期（3-6个月）**：⚖️ **震荡偏强**
- 结汇潮有持续性，能源优势在油价高企时确实提供支撑
- 但需关注中美利差变化和美联储政策动向
- 6.80-6.90区间可能是短期均衡位置

**中期（6-12个月）**：⚠️ **不确定性高**
- 若美联储因通胀反弹转鹰，美元走强可能压制人民币
- 房地产风险和地方债务问题可能重新成为关注点
- 地缘冲突的演变方向难以预测

**中长期（1-3年）**：📉 **不宜过度乐观**
- "石油美元松动"是长期趋势，但过程将非常缓慢
- 新能源出口优势面临贸易保护主义挑战
- 汇率长期走势仍取决于经济基本面和资本账户开放程度

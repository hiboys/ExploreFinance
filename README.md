# Explore Finance Project
ExploreFinance是一个知识分享性项目，源自一个计算机专业出身的证券从业人员，带着『**扎实基础，立足行业，投资未来**』的期许，对金融领域的业余探索。

## 🧱 Architecture

1.**扎实基础**，包括投资学、经济学等学科；

2.**立足行业**，学习行业法律法规、关注监管；

3.**投资未来**，修炼宏观数据分析、财务报表分析和基金投资等技能，建立投资理念和框架。

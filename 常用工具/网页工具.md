# 网页工具

## 股票分析工具

### 公允价值 - GuruFocus
```bash
start "" "https://www.gurufocus.cn/stock/{stockCode}/summary"
```
- example：https://www.gurufocus.cn/stock/000001/summary

### 财联社/CLS搜索
```bash
start "" "https://www.cls.cn/searchPage?keyword={keyword}&type=all"
```

### 巨潮资讯（公告搜索）
```bash
start "" "https://www.cninfo.com.cn/new/fulltextSearch?notautosubmit=&keyWord={keyword}"
```

### 杜邦分析（新浪财经）
```bash
start "" "https://vip.stock.finance.sina.com.cn/corp/go.php/vFD_DupontAnalysis/stockid/{stockCode}/displaytype/10.phtml"
```

### 百度FinScope - 股票分析
```bash
start "" "https://finance.baidu.com/stock/ab-{stockCode}"
```

## ETF/指数分析工具

### 红色火箭基本面
```bash
start "" "https://www.etf818.com/red-rocket/indexDetail?securityCode={stockCode}.{market}"
```
- example: https://www.etf818.com/red-rocket/indexDetail?securityCode=000001.SH

### 百度FinScope - 指数分析
```bash
start "" "https://finance.baidu.com/index/ab-{indexCode}"
```

### 百度FinScope - 板块分析
> blockCode是申万行业代码（最新版）
```bash
start "" "https://finance.baidu.com/block/ab-{blockCode}"
```
- example: https://finance.baidu.com/block/ab-650100 (航天装备Ⅱ)

## 使用说明

### 股票代码格式
- **A股**：直接输入6位数字代码，如 `000001`、`600519`
- **港股**：输入代码+后缀，如 `00700.HK`
- **美股**：输入代码，如 `AAPL`、`TSLA`

### 市场后缀
- `.SH` - 上海证券交易所
- `.SZ` - 深圳证券交易所
- `.HK` - 香港交易所
- `.BJ` - 北京证券交易所

### 申万行业代码
申万行业代码为数字格式，如：
- `650100` - 航天装备Ⅱ
- `801780` - 银行Ⅱ

获取完整行业代码列表可参考 [申万宏源行业分类](https://www.swsresearch.com/institute_sw/allIndex/downloadCenter/industryType)

# 四大维度看AI后续扩散方向——筛选框架深度解析

> **来源**：兴证策略张启尧团队  
> **发布时间**：2026年4月19日  
> **腾讯文档**：https://docs.qq.com/aio/DS1BIRUFrdWFzZWdC

## 一、框架概述

### 1.1 核心逻辑

研报构建了一个**四维交叉验证**的行业筛选框架，用于在AI板块内部寻找**低位绩优+催化确定性**的扩散方向。

```
四大维度 → 筛选条件 → 细分方向矩阵 → 投资建议
```

### 1.2 框架设计原理

| 设计原则 | 说明 |
|---------|------|
| **多维度交叉验证** | 单一维度可能存在偏差，四维交叉提高置信度 |
| **业绩与催化并重** | 既有基本面支撑（业绩），又有事件驱动（催化） |
| **国内与海外映射** | 兼顾国内产业逻辑和全球科技资产共振 |
| **横向与纵向比较** | 既有历史数据（一季报、盈利预期变动），又有空间比较（3月以来涨跌幅） |

## 二、四大维度详解

### 维度一：一季报亮点（业绩验证维度）

#### 2.1.1 筛选逻辑

**核心假设**：一季报是全年业绩的"先验指标"，超预期的细分方向往往具备持续的景气支撑。

**筛选方法**：
- 观察业绩披露后个股的**跳空高开幅度**
- 高开幅度越大 → 超预期程度越高 → 市场认可度越强

#### 2.1.2 筛选结果

| 产业链位置 | 超预期细分方向 |
|-----------|---------------|
| **上游算力硬件&基础设施** | 北美算力（PCB、光模块、液冷）<br>国产算力（存储、国产芯片）<br>AIDC产业链（算力租赁、电力电网） |
| **中下游软件应用** | 网络安全、办公软件<br>游戏、卫星通信、智能家居 |

#### 2.1.3 投资启示

1. **北美算力链**：已形成市场共识，持续凝聚共识，但需警惕成交过热
2. **国产算力&AIDC**：涨幅相对偏低，存在补涨空间
3. **中下游软件**：整体涨幅偏低，细分alpha值得重视

---

### 维度二：年初以来盈利一致预期上修（景气预期维度）

#### 2.2.1 筛选逻辑

**核心假设**：盈利预期变动领先于实际业绩，反映市场对全年景气度的判断。

**筛选方法**：
- 跟踪2026年盈利预测变动情况
- 预期上修 → 景气度改善 → 一季报有望表现较好

#### 2.2.2 筛选结果

**上游算力硬件&基础设施**：
- 光通信（光纤光缆、高速铜缆、光模块、交换机）
- PCB
- 存储芯片
- 北美AIDC（液冷、电源设备）

**中游软件服务**：
- AI编程
- 办公软件

**下游端侧应用**：
- AI应用（金融科技、游戏）
- AI端侧（AI穿戴设备、AI手机）

#### 2.2.3 低位绩优方向（结合3月以来涨跌幅）

| 产业链位置 | 低位绩优方向 |
|-----------|-------------|
| **上游算力硬件** | 国产算力（GPU、半导体材料设备）<br>AIDC产业链（电源设备、电力电网） |
| **中游软件服务** | AI编程、办公软件 |
| **下游端侧应用** | AI端侧（消费电子）<br>AI应用（金融科技、游戏、智慧医疗、数字媒体） |

#### 2.2.4 投资启示

1. **低位+预期上修** = 风险收益比最优的组合
2. 国产算力、AIDC电源设备等方向同时具备"低涨幅+预期上修"特征
3. AI编程、办公软件等中游软件方向值得关注

---

### 维度三：海外映射（全球景气共振维度）

#### 2.3.1 筛选逻辑

**核心假设**：AI是全球性产业趋势，海外科技龙头走势对A股相关板块具有映射指引作用。

**数据基础**：
- 梳理了**40个AI核心细分方向**
- 共**170个海外代表性标的**
- 覆盖美股、港股等全球主要科技市场

#### 2.3.2 筛选结果

**4月8日全球科技资产共振修复以来**：

| 涨幅居前方向 | 代表标的特征 | 逻辑支撑 |
|-------------|-------------|---------|
| **光通信** | 海外光模块、光纤龙头大涨 | 全球算力建设需求旺盛 |
| **存储产业链** | 海外存储芯片龙头强势 | AI驱动存储需求增长 |

**其他映射机会**：
- 云厂商（AWS、Azure、GCP资本开支）
- 智慧医疗
- 游戏
- 智能驾驶
- 卫星通信

#### 2.3.3 投资启示

1. **全球算力景气共振**是本轮科技修复的核心驱动力
2. 光通信、存储等上游硬件是全球资金的共识方向
3. 海外映射可作为A股投资的"先行指标"

---

### 维度四：国内催化（事件驱动维度）

#### 2.4.1 筛选逻辑

**核心假设**：国内重磅事件/产品发布可成为板块行情的催化剂，尤其对相对低位的方向有更强的边际催化作用。

#### 2.4.2 核心催化事件

**DeepSeek V4发布（预期4月下旬）**：

| 突破维度 | 具体亮点 | 受益方向 |
|---------|---------|---------|
| **国产化** | 首次实现与华为昇腾等国产芯片深度适配 | 国产算力、半导体设备 |
| **多模态能力** | 原生支持图片、影音与文本生成 | AI编程、漫短剧、电商、营销 |
| **编程能力** | 代码生成全面优于GPT和Claude | AI编程工具、开发者生态 |

#### 2.4.3 三类扩散机会

基于DeepSeek V4的三重突破，研报识别出三类扩散机会：

**第一类：国产算力**
- 国产芯片龙头
- 半导体材料设备封测环节
- **逻辑**：国产化适配带来需求预期上修

**第二类：国内AIDC产业链**
- 算力租赁（tokens爆发驱动）
- 电力电网等基础设施
- **逻辑**：国产算力需求上修+算力租赁景气度提升

**第三类：多模态领域**
- AI编程
- 漫短剧
- 电商、营销
- **逻辑**：多模态能力突破打开应用场景

#### 2.4.4 投资启示

1. DeepSeek V4发布是**二季度最重要的国内催化**
2. 3月以来涨幅落后的国产算力、中下游软件应用有望迎来补涨
3. 催化兑现前可提前布局，但需警惕"买预期卖事实"风险

## 三、四维框架交叉验证

### 3.1 矩阵分析

将四个维度的筛选结果进行交叉，可得到**高置信度方向**：

| 细分方向 | 一季报 | 预期上修 | 海外映射 | 国内催化 | 综合评分 |
|---------|--------|---------|---------|---------|---------|

| **国产算力** | ✓ | ✓ | △ | ✓✓ | ★★★★ |

## 四、框架应用与局限

### 4.1 框架优势

| 优势 | 说明 |
|------|------|
| **系统性** | 四个维度覆盖业绩、预期、海外、国内，视角全面 |
| **可量化** | 每个维度都有明确的筛选标准和数据来源 |
| **可复制** | 框架可应用于其他主题/板块的行业筛选 |
| **时效性** | 结合季度业绩和事件催化，贴近市场节奏 |

### 4.2 框架局限

| 局限 | 说明 |
|------|------|
| **依赖预期** | 第四维度（国内催化）建立在DeepSeek V4发布的预期上，若延期或不及预期，逻辑受损 |
| **幸存者偏差** | 筛选"超预期"方向时，自然过滤掉表现不佳的板块 |
| **动态变化** | 盈利预期和成交热度是动态变化的，需要持续跟踪更新 |
| **估值约束** | 框架侧重景气度和催化，对估值约束考虑不足 |

### 4.3 使用建议

1. **定期更新**：每季度根据最新业绩和预期数据更新筛选结果
2. **交叉验证**：不要仅依赖单一维度，需多维度交叉确认
3. **估值把关**：在框架筛选基础上，增加估值安全边际判断
4. **催化跟踪**：密切关注核心催化事件（如DeepSeek发布）的进展

## 五、投资决策应用

### 5.1 仓位配置建议

| 方向 | 建议仓位 | 配置逻辑 |
|------|---------|---------|
| **光通信龙头** | 维持现有仓位 | 业绩确定性高，但估值已充分反映 |
| **国产算力** | 10-15% | 四维共振+低位+强催化 |
| **AIDC产业链** | 5-10% | 电源、液冷等环节有业绩支撑 |
| **多模态应用** | <5% | 催化驱动，但业绩兑现不确定 |

### 5.2 风险控制

1. **DeepSeek催化落空风险**：若V4延期或能力不及预期，相关方向可能回调
2. **业绩验证风险**：4月底一季报披露完毕后，需重新评估景气度
3. **估值压缩风险**：光通信等拥挤方向若资金获利了结，可能带动板块调整
4. **海外映射失效风险**：若美股科技股回调，A股映射逻辑可能弱化

### 5.3 关键跟踪指标

- **一季报披露进度**：4月30日前全部披露完毕
- **DeepSeek V4发布进展**：密切关注官方动态
- **光通信成交占比**：监控结构性过热信号
- **海外科技股走势**：跟踪光通信、存储等龙头表现

---

## 六、总结

**四大维度筛选框架**是一个系统性的行业选股工具，通过业绩验证、景气预期、海外映射、国内催化四个维度交叉验证，识别AI板块内部的扩散机会。

**核心结论**：
1. 光通信等龙头方向已形成共识，继续持有但不宜追高
2. 国产算力、AIDC产业链等低位绩优方向具备更高风险收益比
3. DeepSeek V4发布是二季度最重要的国内催化，相关方向可提前布局
4. 需警惕催化落空和估值压缩的风险，做好风险控制
